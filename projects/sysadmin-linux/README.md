# 🐧 Manual SysAdmin Linux

![Status](https://img.shields.io/badge/Estado-Actualizado-green?logo=github)
![Linux](https://img.shields.io/badge/S.O.-Linux-orange?logo=linux)

Manual técnico orientado a la formación progresiva de **Administradores de Sistemas Linux**, desde fundamentos hasta prácticas avanzadas de infraestructura, automatización y contenedores.

---

## Objetivo
Proporcionar una guía **práctica y profesional** para aprender administración Linux con enfoque real de producción, cubriendo buenas prácticas, seguridad y automatización.

Preparado para roles como:
- SysAdmin
- DevOps Engineer
- Site Reliability Engineer (SRE)

---

## Contenido
El manual cubre:

- Fundamentos del sistema y CLI  
- Gestión de usuarios y permisos  
- Administración de paquetes  
- Almacenamiento (LVM, RAID, FS)  
- Systemd y procesos  
- Networking y troubleshooting  
- Seguridad y hardening  
- Servicios (web, bases de datos, SSL)  
- Automatización (Bash, Ansible)  
- Contenedores y virtualización  
- Observabilidad y monitorización  

---

## Requisitos
- VM (VirtualBox, VMware, Proxmox)  
- Distribución recomendada: **Debian / AlmaLinux**  
- LXC / Docker

---

## Enfoque
Cada sección incluye:
- Explicación técnica clara  
- Ejemplos prácticos reales  
- Troubleshooting  
- Tips de entorno profesional  

---

## Tecnologías
Linux · Bash · Systemd · Networking · Docker · Ansible · Prometheus · Grafana

---

## Notas
Este contenido está diseñado como un **itinerario de aprendizaje completo**, orientado a entornos reales de administración e infraestructura.
