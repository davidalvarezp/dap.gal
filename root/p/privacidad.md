## 2. Datos recopilados

Este sitio es estrictamente informativo. El tratamiento de datos personales es mínimo:

- No existen formularios de registro ni formularios de contacto automatizados.
- Se utiliza Google Analytics para estadísticas agregadas de tráfico, **únicamente tras el consentimiento explícito** mediante el banner de cookies. Consulta nuestra política de cookies.
- Si escribes por correo electrónico, los datos que facilites voluntariamente se utilizarán únicamente para responder a tu consulta.

---

## 3. Base jurídica

El tratamiento de los datos analíticos se basa en el **consentimiento** del usuario (art. 6.1.a del RGPD), que puede retirarse en cualquier momento eliminando las cookies o mediante la configuración de privacidad del navegador.

---

## 4. Registros del servidor

La instancia de `nginx` que sirve este sitio registra técnicamente cierta información en sus logs:

- Dirección IP
- `user-agent`
- URL solicitada
- Marca temporal de la petición

Estos registros se utilizan con fines de seguridad (detección de ataques y abusos) y depuración técnica. Se conservan durante un máximo de **30 días** y no se comparten con terceros salvo obligación legal.

---

## 5. Terceros

Este sitio se distribuye a través de **Cloudflare** (CDN / proxy). Cloudflare puede procesar datos técnicos (IP y metadatos de las peticiones) para la protección frente a ataques y la optimización de la entrega del contenido.

Consulta la política de privacidad de Cloudflare:  
https://www.cloudflare.com/privacypolicy/

Si aceptas las cookies analíticas, **Google Analytics** (Google LLC) procesará datos de tráfico anonimizados.

Consulta la política de privacidad de Google:  
https://policies.google.com/privacy
